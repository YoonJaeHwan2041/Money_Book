# 📈 Project MoneyBook: Progress Report (2026-04-08)

## 1. 최근 구현 사항: 카테고리 관리 기능 고도화

### ✅ 대화상자(Dialog) 기반 카테고리 추가 UI 구현
기존의 단순 더미 데이터 삽입 방식에서 벗어나, 사용자가 직접 데이터를 입력하고 관리할 수 있는 **`CategoryAddDialog`**를 구현했습니다.

*   **카테고리명 입력**: `OutlinedTextField`를 통한 실시간 입력.
*   **아이콘 선택 시스템 (4개 옵션)**:
    1.  **이모지(Emoji)**: 사용자가 원하는 이모지 1개를 직접 입력하여 저장.
    2.  **갤러리(Gallery)**: `ActivityResultContracts.PickVisualMedia`를 연동하여 스마트폰 내 사진 선택 가능.
    3.  **기본 리소스(Resource)**: 프로젝트 내 내장 아이콘(`res:ic_...`)을 선택할 수 있는 뼈대 구축.
    4.  **선택 안 함(None)**: 아이콘 없이 기본 폴더 아이콘으로 생성.
*   **이미지 로딩**: `Coil` 라이브러리를 도입하여 갤러리 이미지 및 URI 기반 리소스를 효율적으로 렌더링.

### ✅ 데이터 레이어 및 비즈니스 로직 리팩토링
*   **Hilt DI 확장**: 외부 라이브러리(Coil) 및 이미지 처리 관련 로직 대응.
*   **Room DB 대응**: `iconKey` 필드를 `null` 허용으로 변경하고, 다양한 접두사(`emoji:`, `uri:`, `res:`)를 통해 아이콘 타입을 구분하는 전략 채택.
*   **ViewModel/Repository 업데이트**: 카테고리 추가 시 아이콘 정보를 포함하도록 함수 시그니처 수정.
