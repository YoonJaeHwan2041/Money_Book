# 💰 MoneyBook (가계부 & 스마트 정산)

> **"복잡한 정산까지 한 번에 해결하는 스마트 가계부"**
>
> 단순히 쓰고 읽는 가계부를 넘어, 고정 지출 예약과 친구들과의 복잡한 정산 로직을 한 곳에서 관리할 수 있는 개인 자산 관리 도구입니다.

## ✨ 주요 기능
- **통합 가계부**: 수입, 지출 및 예정된 금융 흐름 관리.
- **스마트 정산**: 1:N 정산 요청 및 실시간 입금 상태 추적.
- **고정 지출 알람**: 반복되는 지출을 잊지 않도록 알림 및 예약 기록.
- **카테고리 커스터마이징**: 아이콘 기반의 자유로운 분류 설정.

## 🛠 기술 스택
- **언어**: Kotlin
- **UI**: Jetpack Compose
- **아키텍처**: MVVM + Clean Architecture (Feature-based)
- **DB**: Room (SQLite)
- **DI**: Hilt
- **비동기**: Coroutines & Flow
